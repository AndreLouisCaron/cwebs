void ws_owire_text ( struct ws_owire *  stream)

Mark the current frame as a text message.

Parameters:
streamThe current writer state.
Warning:
Remember that text messages' payload must be UTF-8 encoded data. This library implements no automatic encoding or decoding of text and it is the application's responsibility to ensure that the payload contains valid UTF-8 data. If the peer strictly enforces the WebSocket specification, it may kill the connection upon receiving invalid UTF-8 data.
